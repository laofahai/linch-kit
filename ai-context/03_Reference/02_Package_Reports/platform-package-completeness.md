## 🎯 包概述

`@linch-kit/platform` 是 LinchKit 的业务开发平台包，整合了原来的 schema、crud、trpc 包的所有功能，提供一体化的业务开发体验。

### 核心职责

- Schema 定义和验证
- CRUD 操作管理
- tRPC API 路由
- 数据验证和转换
- 查询构建器
- 事务管理
